Glosario de Términos Clave

ABCDEFGHIJKLMNOPQRSTUVWXYZ

A

  • A/B Testing
    Método de comparación de dos versiones de una página web o aplicación para determinar cuál rinde mejor en términos de conversión o comportamiento del usuario.
  • Acción (Action) [en GPTs]
    En GPTs personalizados, una “Acción” es una integración externa (API o herramienta) que el asistente puede invocar con parámetros para ejecutar tareas del mundo real (consultas, escrituras, etc.).
  • Agente
    Programa que actúa por vos siguiendo objetivos e instrucciones, como un asistente especializado.
  • Ajuste de Hiperparámetros (Hyperparameter Tuning)
    Proceso de optimización de los parámetros que controlan el comportamiento de un modelo de aprendizaje automático.
  • Algoritmo
    Conjunto de reglas o instrucciones que un sistema sigue para realizar una tarea o resolver un problema específico.
  • Alucinaciones
    Respuestas inventadas o incorrectas que un modelo genera con mucha seguridad.
  • Análisis de Componentes Principales (PCA)
    Técnica de reducción de dimensionalidad que transforma los datos a un conjunto de variables no correlacionadas, llamadas componentes principales.
  • API
    Puerta de acceso para que dos programas se hablen entre sí de forma controlada.
  • Aprendizaje Automático (Machine Learning)
    Subcampo de la IA que permite a los sistemas aprender y mejorar automáticamente a partir de la experiencia sin ser programados explícitamente para ello.
  • Aprendizaje No Supervisado (Unsupervised Learning)
    Tipo de aprendizaje automático en el que el modelo busca patrones en datos sin etiquetas predefinidas.
  • Aprendizaje por Refuerzo (Reinforcement Learning)
    Técnica donde un agente aprende a tomar decisiones mediante prueba y error, recibiendo recompensas o castigos.
  • Aprendizaje Supervisado (Supervised Learning)
    Tipo de aprendizaje automático en el que el modelo se entrena con datos etiquetados.
  • Artificial General Intelligence (AGI) (Inteligencia Artificial General)
    Un tipo teórico de IA que posee la capacidad de entender, aprender y aplicar su inteligencia para resolver cualquier problema que un ser humano pueda.
  • Artificial Intelligence (IA) (Inteligencia Artificial)
    Un campo de la informática centrado en la creación de máquinas que pueden realizar tareas que normalmente requieren inteligencia humana.
  • Artificial Narrow Intelligence (ANI) (Inteligencia Artificial Estrecha)
    El tipo de IA que existe hoy en día. Se especializa en una tarea específica, como jugar al ajedrez o reconocer el habla.
  • Artificial Superintelligence (ASI) (Superinteligencia Artificial)
    Una forma hipotética de IA que no solo imitaría la inteligencia humana, sino que la superaría en todos los aspectos.
  • Autoencoders
    Redes neuronales utilizadas para el aprendizaje no supervisado, que aprenden a comprimir datos (codificador) y luego a reconstruirlos (decodificador).
  • Automatización
    Uso de tecnología para realizar tareas sin intervención humana.

B

  • Bagging
    Técnica de aprendizaje automático que mejora la precisión de un modelo combinando predicciones de varios modelos entrenados en diferentes subconjuntos de datos.
  • Balanceo de Datos
    Técnica para ajustar la distribución de clases en un conjunto de datos para mejorar el rendimiento del modelo.
  • Barreras de seguridad (Guardrails)
    Controles que limitan lo que el sistema puede responder o hacer para evitar usos indebidos.
  • Bias (Sesgo)
    Una tendencia sistemática a favor o en contra de una persona, grupo o idea, que puede ser aprendida y perpetuada por los algoritmos de IA a partir de los datos de entrenamiento.
  • Big Data
    Conjuntos de datos extremadamente grandes y complejos que no pueden ser gestionados o analizados con herramientas de software tradicionales. Son el combustible de la IA moderna.
  • BigQuery
    Data warehouse serverless de Google Cloud para almacenar y consultar grandes volúmenes de datos con SQL estándar y alta escalabilidad.

C

  • Callback URL / Redirect URI (URL de callback)
    Dirección registrada a la que el proveedor de autenticación redirige al usuario con un código o token tras completar el login (p. ej., en OAuth).
  • Chatbot
    Programa que simula una conversación humana, respondiendo preguntas y realizando tareas específicas.
  • Clasificación
    Tarea de asignar una etiqueta a un dato entre varias opciones posibles.
  • Client ID
    Identificador público de una aplicación en un flujo de autenticación.
  • Clustering (Agrupamiento)
    Agrupamiento de datos similares en categorías o clusters.
  • Computación en la Nube (Cloud Computing)
    Uso de servidores remotos en Internet para almacenar, gestionar y procesar datos en lugar de utilizar servidores locales.
  • Concurrencia (Concurrency)
    Capacidad de ejecutar múltiples tareas de forma solapada (no necesariamente en paralelo), compartiendo recursos de manera segura.
  • Conjunto de Datos (Dataset)
    Colección de datos utilizados para entrenar o evaluar modelos de IA.
  • Contexto
    Información que recibe el modelo para entender tu pedido y responder mejor.
  • Conversación
    Intercambio de mensajes con el modelo que mantiene un hilo y memoria de lo anterior.
  • Convolución
    Operación matemática utilizada en redes neuronales convolucionales para procesar datos estructurados en forma de cuadrículas (imágenes).
  • Convolutional Neural Networks (CNNs) (Redes Neuronales Convolucionales)
    Un tipo de red neuronal profunda que se utiliza principalmente para analizar imágenes visuales. Son la base del reconocimiento de imágenes y la visión por computadora.
  • Costo por token
    Lo que se paga por cada unidad de texto procesada por el modelo.
  • CSRF
    Ataque que fuerza a un usuario autenticado a ejecutar acciones no deseadas en una app web. Se mitiga con tokens anti‑CSRF, SameSite en cookies, y verificación de origen.
  • CSV
    Formato de archivo de texto donde cada fila es un registro y las columnas están separadas por comas.
  • CTR (Click-Through Rate)
    Porcentaje de clics sobre impresiones en campañas o emails. Se calcula como (clics / impresiones) × 100.
  • Curva ROC (ROC Curve)
    Gráfico que muestra el rendimiento de un modelo de clasificación para diferentes umbrales de decisión.

D

  • Data Mining
    Proceso de descubrir patrones y conocimiento a partir de grandes cantidades de datos.
  • Data Warehouse
    Repositorio centralizado y optimizado para análisis de datos históricos del negocio, separado de los sistemas transaccionales.
  • Deadlock
    Bloqueo mutuo entre procesos o hilos que esperan recursos entre sí y ninguno puede continuar.
  • Decision Trees (Árboles de Decisión)
    Un algoritmo de aprendizaje supervisado que utiliza una estructura similar a un árbol para tomar decisiones basadas en las características de los datos.
  • Deep Learning (Aprendizaje Profundo)
    Rama del aprendizaje automático que utiliza redes neuronales profundas para modelar datos complejos.
  • Desbalance de Clases (Class Imbalance)
    Situación en la que una clase está representada mucho más que otras en un conjunto de datos, lo que puede afectar el rendimiento del modelo.
  • Determinismo
    Que una misma entrada siempre produce la misma salida (cuando no hay azar).
  • Diffusion Models (Modelos de Difusión)
    Una clase de modelos generativos que crean datos, como imágenes, añadiendo y luego eliminando progresivamente "ruido" de una muestra inicial.
  • Difusión
    En IA generativa, familia de modelos que crean imágenes u otros datos agregando y quitando “ruido”.
  • Dimensionality Reduction (Reducción de Dimensionalidad)
    El proceso de reducir el número de variables de entrada en un conjunto de datos, para simplificar el modelo y mejorar su rendimiento.

E

  • Ecosistema
    Conjunto de herramientas y servicios que se integran entre sí.
  • Embedding
    Representaciones vectoriales de datos, como palabras o imágenes, en un espacio de características continuas.
  • Embeddings
    Representaciones numéricas de textos o imágenes para poder compararlos por su “significado”.
  • Endpoint
    Dirección específica de una API a la que se le hacen pedidos.
  • Ethics (Ética)
    El estudio de los principios morales y los valores que guían el desarrollo y el uso de la tecnología de IA de manera responsable.
  • Evaluación Cruzada (Cross-Validation)
    Técnica para evaluar el rendimiento de un modelo dividiendo los datos en múltiples subconjuntos.
  • Exfiltración de datos
    Cuando alguien intenta que el sistema revele información privada o sensible sin permiso.
  • Exploración de Datos
    Proceso de analizar y resumir las principales características de un conjunto de datos, a menudo con métodos visuales.

F

  • FAQ (Preguntas Frecuentes)
    Listado de preguntas y respuestas comunes para resolver dudas rápidas de usuarios.
  • Feature (Característica)
    Una variable o atributo individual y medible de los datos que se utiliza como entrada para un modelo de Machine Learning.
  • Feature Engineering (Ingeniería de Características)
    Proceso de seleccionar, modificar o crear variables que representen mejor los datos para mejorar el rendimiento de un modelo.
  • Few-shot
    Dar pocos ejemplos en el prompt para que el modelo copie el estilo o el formato.
  • Fine‑tuning (Ajuste fino)
    Entrenar el modelo con tus propios ejemplos para que se adapte mejor a tu uso.
  • Framework
    Una plataforma o biblioteca de software que proporciona una base para desarrollar aplicaciones de software, como los frameworks de Deep Learning (TensorFlow, PyTorch).
  • Función de Activación
    Función matemática utilizada en una red neuronal para decidir si una neurona debe activarse o no.

G

  • GCP (Google Cloud Platform)
    Suite de servicios en la nube de Google: cómputo, almacenamiento, BigQuery, redes y más.
  • Generación de Lenguaje Natural (NLG)
    Proceso de generar texto coherente y comprensible en lenguaje natural a partir de datos estructurados.
  • Generative Adversarial Networks (GANs) (Redes Generativas Antagónicas)
    Una arquitectura de Deep Learning que consta de dos redes neuronales, un generador y un discriminador, que compiten entre sí para crear datos sintéticos realistas.
  • Generative AI (IA Generativa)
    Una rama de la IA que se centra en la creación de contenido nuevo y original, como texto, imágenes o música.
  • GET (método HTTP)
    Método idempotente para obtener recursos del servidor sin producir efectos secundarios en el estado del sistema.
  • GPTs
    Versiones personalizadas de GPT que siguen reglas y usan herramientas específicas.
  • GPU (Unidad de Procesamiento Gráfico)
    Hardware especializado en realizar operaciones matemáticas intensivas, comúnmente utilizado en Deep Learning.
  • Gradiente Descendente
    Algoritmo de optimización utilizado para minimizar la función de error de un modelo ajustando los parámetros.
  • Grid Search
    Método exhaustivo de ajuste de hiperparámetros que prueba todas las combinaciones posibles para encontrar la mejor configuración.

H

  • HTML
    Lenguaje con el que se estructura el contenido de las páginas web.
  • HTTPS
    Versión segura de HTTP que cifra la comunicación con TLS/SSL, garantizando confidencialidad e integridad.

I

  • IA Generativa
    Subcampo de la IA que se enfoca en la creación de nuevos contenidos, como imágenes o texto, a partir de modelos entrenados.
  • Idempotencia
    Propiedad de una operación que, al repetirse con el mismo input, produce el mismo resultado sin efectos adicionales (p. ej., GET, PUT).
  • Inferencia
    Proceso de usar un modelo entrenado para hacer predicciones sobre nuevos datos.
  • Inteligencia Artificial (IA)
    Campo de la informática que se dedica a crear sistemas capaces de realizar tareas que normalmente requieren inteligencia humana.
  • Interfaz
    Forma en que te comunicás con una herramienta o sistema.
  • Inyección de prompts (Prompt Injection)
    Texto malicioso que busca cambiar las instrucciones del asistente para que haga algo no esperado.

J

  • Jailbreak
    Truco de mensajes para saltar o engañar las protecciones del modelo y lograr respuestas que no debería dar.
  • JSON
    Formato de texto para intercambiar datos: usa pares clave–valor y listas, fácil de leer por personas y programas.
  • JSONL
    Formato donde cada línea es un objeto JSON; muy usado para entrenar modelos con muchos ejemplos.

K

  • K-Means
    Algoritmo de agrupamiento que divide los datos en K clusters, minimizando la distancia de cada punto al centro del cluster.
  • K-Vecinos Más Cercanos (K-Nearest Neighbors)
    Algoritmo de clasificación que asigna una etiqueta a un dato basándose en las etiquetas de sus vecinos más cercanos.
  • Keras
    Una API de alto nivel para construir y entrenar modelos de Deep Learning, que se ejecuta sobre frameworks como TensorFlow.
  • KPI (Key Performance Indicator)
    Métrica clave para evaluar el desempeño respecto a un objetivo.

L

  • Labeled Data (Datos Etiquetados)
    Conjunto de datos que incluye tanto las características de entrada como las etiquetas de salida, utilizadas para el aprendizaje supervisado.
  • Large Language Models (LLMs) (Grandes Modelos de Lenguaje)
    Modelos de Deep Learning entrenados con enormes cantidades de datos de texto, capaces de comprender y generar lenguaje natural.
  • Lasso
    Tipo de regresión que agrega una penalización a la suma de los valores absolutos de los coeficientes, ayudando a la selección de características.
  • Latencia
    Tiempo que tarda en aparecer la respuesta desde que enviás la solicitud.
  • Lazy loading
    Estrategia para cargar recursos o datos solo cuando se necesitan, mejorando el tiempo de carga y el uso de memoria.
  • Learning Rate (Tasa de Aprendizaje)
    Parámetro que controla qué tan rápido un modelo ajusta sus parámetros durante el entrenamiento.
  • Linear Regression (Regresión Lineal)
    Un algoritmo de aprendizaje supervisado que se utiliza para predecir un valor numérico continuo basándose en una relación lineal entre las variables.
  • LLM (Large Language Model)
    Modelos de lenguaje a gran escala, como GPT, entrenados con grandes cantidades de texto para realizar diversas tareas de procesamiento del lenguaje natural.
  • Logistic Regression (Regresión Logística)
    Un algoritmo de aprendizaje supervisado que se utiliza para tareas de clasificación, prediciendo la probabilidad de que una entrada pertenezca a una categoría.
  • Long Short-Term Memory (LSTM)
    Un tipo de arquitectura de Red Neuronal Recurrente (RNN) diseñada para superar los problemas de memoria a corto plazo de las RNN tradicionales.

M

  • Markdown
    Sintaxis simple para dar formato a texto (títulos, listas, enlaces).
  • Matriz de Confusión
    Tabla utilizada para evaluar el rendimiento de un modelo de clasificación, mostrando los verdaderos positivos, falsos positivos, verdaderos negativos y falsos negativos.
  • Model (Modelo)
    La representación matemática de un sistema que se utiliza para hacer predicciones o decisiones basadas en datos.
  • Modelado
    Proceso de crear, entrenar y validar un modelo de IA para resolver un problema específico.
  • Modelo de lenguaje
    Programa entrenado para leer y generar texto de forma coherente.
  • Moderación
    Proceso de revisar y bloquear contenido que viola las reglas.
  • Multimodal
    Capacidad de trabajar con varios tipos de datos: texto, imágenes, audio, etc.

N

  • Natural Language Processing (NLP) (Procesamiento de Lenguaje Natural)
    Un campo de la IA que se ocupa de la interacción entre las computadoras y el lenguaje humano.
  • Navegación
    Recorrido por las secciones de un sitio o app.
  • Neural Networks (Redes Neuronales)
    Modelos computacionales inspirados en el cerebro humano, compuestos por nodos interconectados (neuronas) que procesan y transmiten información.
  • Neurona
    Unidad básica de una red neuronal, que recibe entradas, las procesa y produce una salida.
  • Newsletter
    Boletín informativo periódico enviado por email a una lista de suscriptores.
  • Normalización
    Proceso de ajustar los valores de un conjunto de datos para que se encuentren en una escala común.

O

  • OAuth 2.0
    Estándar de autorización para obtener acceso delegado a recursos protegidos sin compartir credenciales del usuario con la aplicación cliente.
  • OCR
    Tecnología que convierte una imagen o PDF en texto editable.
  • OpenAI
    Organización que desarrolla modelos como GPT y herramientas relacionadas.
  • OpenAPI (esquema)
    Especificación para describir APIs HTTP de forma legible por humanos y máquinas (endpoints, parámetros, esquemas y respuestas).
  • Optimización
    Proceso de ajustar un modelo para mejorar su rendimiento en una tarea específica.
  • Overfitting (Sobreajuste)
    Situación en la que un modelo se ajusta demasiado a los datos de entrenamiento y no generaliza bien en datos nuevos.

P

  • API Key (Clave API)
    Código secreto que te identifica cuando usás una API.
  • Pandas
    Biblioteca de Python utilizada para manipulación y análisis de datos.
  • Parámetro
    Valores internos de un modelo que se ajustan durante el entrenamiento para minimizar el error en las predicciones.
  • PDF
    Formato de documento que conserva el diseño al compartirlo.
  • Penalización de frecuencia
    Ajuste que reduce repeticiones cuando el modelo insiste con las mismas palabras.
  • Penalización de presencia
    Ajuste que empuja al modelo a mencionar ideas nuevas en lugar de repetir.
  • Perceptrón
    Modelo de neurona artificial que es la base de las redes neuronales.
  • PHI (Información de salud protegida)
    Datos de salud personales que deben tratarse con especial cuidado.
  • PII (Datos personales identificables)
    Información que puede identificar a una persona: nombre, DNI, email, teléfono, etc.
  • Pipeline
    Serie de pasos o procesos por los que pasan los datos desde su entrada hasta la salida del modelo.
  • Plataforma de Código Abierto (Open Source)
    Software cuyo código fuente está disponible para que cualquiera lo use, modifique y distribuya.
  • Plataforma de Código Cerrado (Closed Source)
    Software cuyo código fuente no está disponible públicamente y suele ser propiedad de una empresa.
  • PNG
    Formato de imagen de buena calidad y fondo transparente.
  • Ponderación
    Asignación de importancia relativa a diferentes características o entradas en un modelo de IA.
  • POST (método HTTP)
    Método para enviar datos al servidor, crear recursos o ejecutar operaciones con efectos secundarios; no es idempotente.
  • Precisión
    Medida que indica el porcentaje de predicciones correctas realizadas por un modelo.
  • Preferencias
    Configuración elegida por el usuario para personalizar la experiencia.
  • Preprocesamiento
    Conjunto de técnicas para preparar los datos antes de ser utilizados en un modelo de IA.
  • Principal Component Analysis (PCA) (Análisis de Componentes Principales)
    Una técnica de reducción de dimensionalidad que transforma los datos en un nuevo conjunto de variables (componentes principales) no correlacionadas.
  • Privacidad
    Protección de tus datos personales y de cómo se usan.
  • Prompt
    Entrada de texto que se da a un modelo de lenguaje, como GPT, para que genere una respuesta o complete una tarea.
  • Prompt Engineering
    Diseñar instrucciones claras y ejemplos para obtener mejores resultados.
  • Prompting
    El proceso de diseñar y refinar las instrucciones (prompts) que se le dan a un modelo de IA generativa para obtener los resultados deseados.
  • Propagación hacia Atrás (Backpropagation)
    Algoritmo utilizado en el entrenamiento de redes neuronales para ajustar los pesos en función del error de las predicciones.
  • Python
    Lenguaje de programación de alto nivel, legible y multiparadigma, muy usado en ciencia de datos, automatización y desarrollo web.
  • PyTorch
    Un popular framework de Deep Learning de código abierto, conocido por su flexibilidad y facilidad de uso en la investigación.

Q

  • Q-Learning
    Un algoritmo de aprendizaje por refuerzo que aprende una política de acción óptima, indicando a un agente qué acción tomar en cada estado.
  • QA (Quality Assurance)
    Prácticas para asegurar la calidad del software: pruebas, revisiones y estándares antes de publicar.

R

  • RAG (Retrieval-Augmented Generation)
    Patrón que combina recuperación de información y generación con LLMs.
  • Random Forest
    Algoritmo de aprendizaje automático que utiliza múltiples árboles de decisión para mejorar la precisión de las predicciones.
  • Razonamiento paso a paso (Chain‑of‑Thought)
    Pedirle al modelo que piense en varios pasos simples en lugar de ir directo a la respuesta.
  • Red Neuronal
    Modelo de IA inspirado en el funcionamiento del cerebro humano, compuesto por capas de neuronas artificiales.
  • Red Neuronal Convolucional (CNN)
    Tipo de red neuronal especializada en el procesamiento de datos con estructura en forma de cuadrícula, como imágenes.
  • Red Neuronal Recurrente (RNN)
    Tipo de red neuronal que es especialmente útil para procesar datos secuenciales como texto o series temporales.
  • Red Teaming
    Pruebas de ataque controladas para encontrar fallas de seguridad antes de que lo hagan los malos.
  • Regresión
    Tarea de predicción de un valor continuo, como la temperatura o el precio de una vivienda.
  • Regularización
    Técnicas utilizadas para evitar el sobreajuste de un modelo, como la penalización de grandes coeficientes.
  • Repositorio
    Lugar donde se guardan versiones de archivos y código.
  • Representación
    Forma en que los datos o características son almacenados y procesados en un modelo de IA.
  • ROI (Return on Investment)
    Indicador que mide la ganancia obtenida en relación con lo invertido.

S

  • Sampleo
    Proceso de seleccionar un subconjunto de datos de un conjunto más grande.
  • SARSA
    Un algoritmo de aprendizaje por refuerzo similar a Q-Learning, que también aprende una política de acción óptima.
  • Scope (OAuth)
    Ámbito o permisos solicitados en OAuth que limitan qué recursos puede acceder la aplicación.
  • Script
    Archivo de instrucciones (p. ej., .py, .sh) ejecutado por un intérprete para automatizar tareas.
  • SDK
    Paquete de herramientas para programar más fácil con un servicio o plataforma.
  • Segmentación
    Proceso de dividir un conjunto de datos en segmentos o grupos más pequeños que comparten características similares.
  • Semi-Supervised Learning (Aprendizaje Semi-Supervisado)
    Un enfoque de aprendizaje que utiliza una combinación de datos etiquetados y no etiquetados para entrenar un modelo.
  • Semáforo (programación concurrente)
    Primitiva de sincronización que controla el acceso concurrente a recursos mediante contadores.
  • Semántica
    Significado de las palabras y cómo se relacionan en un texto.
  • Sentiment Analysis (Análisis de Sentimientos)
    Técnica de procesamiento de lenguaje natural para identificar y extraer opiniones o sentimientos de un texto.
  • Sesión
    Período en que estás conectado y usando un sistema.
  • SLA (Service Level Agreement)
    Compromiso formal de niveles de servicio (p. ej., disponibilidad y tiempos de respuesta) entre proveedor y cliente.
  • Speech-to-Text
    Tecnología que convierte la voz humana en texto escrito.
  • SQL
    Lenguaje estándar para consultar y manipular datos en bases de datos relacionales (SELECT, INSERT, UPDATE, DELETE).
  • Stacktrace
    Secuencia de llamadas activa al producirse un error o excepción; muy útil para depurar.
  • Subajuste (Underfitting)
    Situación en la que un modelo es demasiado simple para capturar los patrones en los datos, resultando en un bajo rendimiento.
  • Subprocess (módulo / procesos hijo)
    Mecanismo (p. ej., en Python) para crear procesos externos y comunicarse con ellos (stdin/stdout/stderr).
  • Support Vector Machine (Máquina de Vectores de Soporte)
    Algoritmo de clasificación que busca un hiperplano que mejor separe las clases en los datos.
  • SVG
    Formato de imagen vectorial que no pierde calidad al escalarse.

T

  • Temperatura
    Ajuste que controla cuán creativas o conservadoras son las respuestas.
  • Tensor
    Estructura de datos multidimensional utilizada en el procesamiento y el modelado de datos.
  • TensorFlow
    Framework de código abierto para construir y entrenar modelos de aprendizaje automático y aprendizaje profundo.
  • Token
    Unidad básica de texto, como una palabra o carácter, que se utiliza en modelos de lenguaje.
  • Tokenización
    Proceso de convertir un texto en unidades mínimas (tokens).
  • Traducción
    Pasar de un idioma a otro manteniendo el sentido.
  • Training (Entrenamiento)
    El proceso de enseñar a un modelo de Machine Learning a realizar una tarea dándole ejemplos de datos.
  • Transfer Learning (Aprendizaje por Transferencia)
    Técnica donde un modelo preentrenado en una tarea se adapta a otra tarea relacionada.
  • Transformers
    Una arquitectura de red neuronal que se ha convertido en el estándar para tareas de NLP, utilizando un mecanismo de "atención" para procesar secuencias de datos.
  • Tree of Thoughts (ToT)
    Técnica de prompting que explora múltiples líneas de razonamiento como un árbol, evalúa alternativas y elige la mejor rama.
  • TTL (Time To Live)
    Tiempo de vida de un dato o caché (o registro DNS) antes de expirar y renovarse.

U

  • UI
    Parte visual con la que interactuás: botones, menús, formularios.
  • URL
    Dirección única de una página o recurso en internet.

V

  • Validación
    Proceso de evaluar el rendimiento de un modelo en un conjunto de datos diferente al utilizado para el entrenamiento.
  • Variables de entorno (.env)
    Pares clave‑valor definidos en el sistema o en un archivo .env para configurar credenciales y parámetros sin hardcodearlos.
  • Vector
    Arreglo unidimensional de números, utilizado para representar datos en modelos de IA.
  • Ventana de contexto
    Máxima cantidad de texto que el modelo puede tener en mente a la vez.
  • Versionado
    Control de cambios en archivos o proyectos a lo largo del tiempo.
  • Visualización de Datos
    Proceso de representar datos en formato visual para facilitar su comprensión.

W

  • Word Embedding
    Representación vectorial de palabras en un espacio de características, utilizado en procesamiento de lenguaje natural.

X

  • XGBoost
    Algoritmo de aprendizaje automático basado en árboles de decisión que es conocido por su alta precisión y eficiencia.
  • XSS (Cross-Site Scripting)
    Vulnerabilidad que inyecta scripts en páginas vistas por otros usuarios.

Y

  • YOLO (You Only Look Once)
    Algoritmo de detección de objetos en imágenes que es rápido y preciso.

Z

  • Zero-shot Learning
    Técnica en la que un modelo es capaz de reconocer y clasificar datos de clases que no ha visto durante el entrenamiento.
